# Site de Manutenção de Informática

Para abrir o site localmente:

1. No Windows Explorer, navegue até a pasta `João Paulo site`.
2. Dê um duplo clique em `run-site.bat`.

Isso abrirá `index.html` no navegador padrão.

## Alternativa

Se você quiser servir o site em um servidor local, use o Python (se estiver instalado):

```powershell
cd "c:\Users\Jenifer Schwanck\Desktop\João Paulo site"
python -m http.server 8000
```

Depois abra `http://localhost:8000` no navegador.
